# E-Commerce Sales & RFM Analytics System

## Overview
A cloud-based full-stack sales management and analytics platform designed to manage transactions, track performance, and generate customer insights using the RFM (Recency, Frequency, Monetary) model.

The system supports multiple user roles and provides real-time dashboards for data-driven business decisions.

## Features
- Multi-role login: Admin, Clerk, Stakeholder
- Secure password hashing and authentication
- Sales entry with transactional integrity
- Automated order, payment, and audit logging
- RFM analytics dashboard
- Revenue visualization and customer insights
- Cloud deployment with PostgreSQL
- Role-based access control

## Tech Stack
- Streamlit (Frontend)
- Python + SQLAlchemy (Backend)
- PostgreSQL (Database)
- Render (Deployment)
- Pandas (Data Processing)
